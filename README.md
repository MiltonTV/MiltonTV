# Milton TV

Milton TV es una lista IPTV gratuita y legal diseñada para funcionar con TiviMate, Google TV, Android TV y otros reproductores compatibles con M3U.

## Objetivos

- Canales abiertos de Colombia.
- Noticias internacionales.
- Cultura y educación.
- Canales FAST gratuitos.
- Organización por categorías.
- Compatibilidad con EPG.

## Versiones

### MiltonTV Lite
Versión inicial con canales esenciales.

### MiltonTV v1.0
Versión estable con aproximadamente 50 canales gratuitos y legales.

## Estado del proyecto

🚧 En desarrollo.

## Instalación en TiviMate

Cuando MiltonTV esté publicado, utiliza las siguientes direcciones:

### Playlist

```text
https://raw.githubusercontent.com/MiltonTV/MiltonTV/main/playlists/MiltonTV.m3u
```

### EPG

```text
https://raw.githubusercontent.com/MiltonTV/MiltonTV/main/epg/MiltonTV.xml
```

> Actualmente el proyecto está en desarrollo.
